# Horizontal Align

When [HorizontalScreenCoverage](/settings/horizontal-screen-coverage) is below ```100```, this setting determines where the terminal is place horizontally.

Possible values:

- Left
- Center (default)
- Right

```json
{
	"HorizontalAlign": "Center"
}
```
